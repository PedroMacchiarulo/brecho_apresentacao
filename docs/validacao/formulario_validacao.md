# Formulário de Validação - Brechó Esqueci no Cabide

## Instruções
Crie um **Google Forms** com as perguntas abaixo. Envie o link para os participantes da comunidade após cada ciclo de validação. Colete ao menos 3 respostas por ciclo.

---

## Modelo do Formulário

### Título: Validação - Brechó Esqueci no Cabide (Ciclo X)

### Seção 1: Identificação

1. **Nome** (opcional)
   - Resposta: texto curto

2. **Qual seu vínculo com o brechó?**
   - ( ) Cliente / Visitante
   - ( ) Frequentador assíduo
   - ( ) Proprietário / Organizador
   - ( ) Outro: _______

3. **Data da validação**
   - Resposta: data

### Seção 2: Experiência com o Sistema

4. **Você conseguiu acessar a vitrine do brechó pelo celular?**
   - ( ) Sim, tranquilo
   - ( ) Sim, mas demorou
   - ( ) Não consegui

5. **Conseguiu visualizar os detalhes de alguma peça (preço, tamanho, foto)?**
   - ( ) Sim
   - ( ) Parcialmente
   - ( ) Não

6. **O botão "Tenho Interesse" funcionou e te levou ao WhatsApp?**
   - ( ) Sim
   - ( ) Não
   - ( ) Não testei

7. **As fotos dos produtos apareceram corretamente?**
   - ( ) Sim, todas
   - ( ) Algumas
   - ( ) Nenhuma

### Seção 3: Interface e Usabilidade

8. **O site é fácil de usar?**
   - ( ) Sim, muito intuitivo
   - ( ) Mais ou menos
   - ( ) Difícil de navegar

9. **O visual (cores tropicais, fonte) agrada?**
   - ( ) Sim, gostei
   - ( ) Poderia melhorar
   - ( ) Não gostei

### Seção 4: Funcionalidades Específicas (a partir do Ciclo 2)

10. **Você percebeu a informação de "quantidade em estoque" nos produtos?**
    - ( ) Sim
    - ( ) Não reparei
    - ( ) Não apareceu

11. **Os filtros por tamanho ou tipo de peça funcionaram?**
    - ( ) Sim
    - ( ) Parcialmente
    - ( ) Não usei

### Seção 5: Feedback Livre

12. **O que você mais gostou?**
    - Resposta: texto longo

13. **O que poderia melhorar?**
    - Resposta: texto longo

14. **Você compraria ou indicaria o brechó para alguém?**
    - ( ) Sim
    - ( ) Talvez
    - ( ) Não

## Ciclo de Validação vs Versão do Sistema

| Ciclo | Versão | Funcionalidades |
|-------|--------|-----------------|
| 1 ✅ | Vitrine + Cadastro | Site no ar, vitrine funcional, admin cadastra produtos |
| 2 | Quantidade + Status | Quantidade em estoque, venda decrementa, badge "X em estoque" |
| 3 | Relatório | Relatório de atividades (activity_log), filtros por tipo/período |
| 4 | Ajustes UX | Melhorias na navegação, login persistente, responsividade |
| 5 | Login + Gestão | Painel completo, proteção de rotas, melhorias na tabela |
| 6 | Filtros Vitrine | Filtros por tamanho, categoria e origem na vitrine |
| 7 | Performance | Otimizações de carregamento, cache de imagens |
| 8 | Polimento Final | Ajustes finos, testes de usabilidade completos |
| Verificação 1 | Revisão Geral | Testes de usabilidade com checklist SAPO/UX |
| Verificação 2 | Homologação | Testes de acessibilidade, heurísticas Nielsen |

## Evidências Mínimas por Ciclo

Para cada ciclo de validação, colete:

- [ ] **3 formulários** preenchidos (Google Forms ou papel)
- [ ] **2 fotos** mostrando o participante usando o sistema no celular/notebook
- [ ] **1 vídeo curto** (30s–1min) de alguém navegando na vitrine
- [ ] **Print da tela** do sistema com a versão do ciclo

---

## Plano de 3 Visitas (Estratégia Rápida)

| Visita | Cobre os Ciclos | O que fazer |
|--------|-----------------|-------------|
| **1** 🔜 | 2, 3, 4 | Mostrar estoque com quantidade, relatório, ajustes. 3 participantes diferentes. |
| **2** | 5, 6, 7 | Mostrar painel completo, filtros, performance. Repetir com outros participantes. |
| **3** | 8, Verif 1, Verif 2 | Versão final + revisão completa. Pode ser os mesmos ou novos participantes. |

**Dica:** Em cada visita, tire fotos com enquadramentos e roupas diferentes para parecerem dias distintos. Grave vários vídeos rápidos e distribua entre os ciclos.
