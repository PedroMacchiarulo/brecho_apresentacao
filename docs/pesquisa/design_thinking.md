# Pesquisa: Design Thinking (DT)

## 1. O que é?
Design Thinking é uma abordagem centrada no ser humano para a resolução de problemas. Não é apenas sobre a interface, mas sobre entender a dor do usuário.

## 2. Etapas Aplicadas ao Projeto
- **Imersão (Empatia):** Conversas com a comunidade e donos do "Esqueci no Cabide" para entender as dificuldades de venda.
- **Ideação:** Brainstorming de funcionalidades (ex: etiquetas de outlet, categorias tropicais).
- **Prototipação:** Criação do MVP simples em HTML/CSS/JS.
- **Teste/Validação:** Coleta de feedback real dos usuários.

**Conclusão:** O uso de DT garante que o software não seja apenas funcional, mas que resolva um problema real da comunidade local.
