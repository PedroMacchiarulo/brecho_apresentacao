# Pesquisa: Modelo 3C de Colaboração

## 1. Os Três Pilares (3Cs)
Para que o e-commerce seja realmente "colaborativo", ele deve contemplar:
- **Comunicação:** Meios para o comprador e vendedor interagirem (ex: botão de interesse que leva ao WhatsApp).
- **Coordenação:** Organização de quem vende, quem compra e a categoria da peça (Brechó vs Outlet).
- **Cooperação:** A comunidade trabalhando junta para promover a moda sustentável, onde o vendedor ajuda o comprador a encontrar a peça ideal.

## 2. Mecanismo de Percepção
É a capacidade do usuário de entender rapidamente o estado do sistema. No nosso caso, as **Badges coloridas** (Verde/Amarelo) servem como esse mecanismo, informando instantaneamente a origem da peça.

**Conclusão:** A colaboração não é apenas "vender", mas criar um ecossistema de troca consciente.
